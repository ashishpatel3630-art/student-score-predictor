# 🎓 Student Score Prediction ML Project

## 📌 Objective
Predict student final scores based on study hours, attendance, and previous marks.

## 📊 Dataset
Synthetic dataset with 1000+ rows.

## 🧠 Model Used
- Linear Regression

## 📈 Results
- R² Score: 0.89
- MAE: 3.8

```bash
pip install -r requirements.txt
python student_model.py
